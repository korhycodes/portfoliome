# DevDesigner - Portfolio
This repository hosts the source code for a personal portfolio website titled "DevDesigner Portfolios". The site is designed to showcase projects and provide contact information for potential employers or collaborators.

## Technologies Used
- **HTML5**: For the structure of the website.
- **Tailwind CSS**: A utility-first CSS framework for styling. The configuration in the index.html file shows custom colors (primary, background-light, background-dark) and font families (Inter).
JavaScript: Used for functionality, potentially for the animations or interactive elements suggested by the video files (Burger Animation.mp4, Pizzamania.mp4).

### Project Structure
- *The key files in the repository include*:
- **Home Page**: The main landing page.
- **About Page**: Information about the designer.
- **Work Page**: A page dedicated to showcasing work/projects.
- **README.md**: This documentation file.
- **Media**: Image assets used on the site and Video assets for project demonstrations or animations.

## Installation
- **To run this project locally**:
1. Clone the repository:
```bash
   git clone <repository-url>
   ```
2. Navigate to the project directory:

 cd <project-name>

3. Open the website:
Simply open the index.html file in your preferred web browser. No complex build process or server-side dependencies are incuded.

The website is a static portfolio. Navigate through the different pages using the links provided (about.html, work.html) to view the designer's profile and projects.
